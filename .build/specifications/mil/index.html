Machine Interface Language

Grammar

Module

A module consists of zero or more class, enumeration, and procedure definitions.

module : definition*
definition : classDefinition
           | procedureDefinition

Class Definition

classDefinition : 'class' name ('extends' name)? classBody
procedureBody : '{' classMemberDefinition '}'
classMemberDefinition : constructorDefinition
                      | methodDefinition
                      | fieldDefinition

Constructor Definition

constructorDefinition : 'constructor' ('native' string)? constructorParameters
                      | 'constructor' constructorParameters constructorBody
constructorBody : '{' statements '}'

Method Definition

methodDefinition : 'method' ('native' string)? name methodParameters
                 | 'method' methodParameters methodBody
methodBody : '{' statements '}'

Procedure Definition

procedureDefinition : 'procedure' 'entry'? ('native' string)? name procedureParameters
                    | 'procedure' 'entry'? name procedureParameters procedureBody
procedureBody : '{' statements '}'

Instruction Statement

An instruction statement is an instruction.

instructionStatement : instruction

Definition Statement

A definition statement is a variable definition statement or a label definition statement.

definitionStatement : variableDefinitionStatement
                    | labelDefinitionStatement

Label Definition Statement

A label definition statement is a name followed by a colon.

labelDefinitionStatement : name ':'

Variable Definition Statement

A variable definition statement is the variable keyword followed by the name of the variable.

variableDefinitionStatement : 'variable' name
