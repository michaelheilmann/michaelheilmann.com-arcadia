Michael Heilmann's Arcadia Math

This is the documentation for Michael Heilmann's Arcadia Math. Arcadia Math provides matrix and vector math used in the fields of audials and visuals.

Files

You can find the sources of Arcadia Math in my GitHub repository https://github.com/michaelheilmann/michaelheilmann.com. The subdirectory of Arcadia Math in the repository is here https://github.com/michaelheilmann/michaelheilmann.com/tree/main/repository/Math.

  • The (sources of the) library resides in the directory https://github.com/michaelheilmann/michaelheilmann.com/tree/main/repository/Math/Library
  • The (sources of) tests reside in the directory https://github.com/michaelheilmann/michaelheilmann.com/tree/main/repository/Math/Tests
  • The (sources of this very documentation you are reading) documentation reside in the directory https://github.com/michaelheilmann/michaelheilmann.com/tree/main/repository/Math/Documentation

Arcadia Math supports various platforms (including but not restricted to Windows, Linux, and many more), however, we currently only officially support Windows. For instructions on how to build, test, and use Arcadia Math, refer to README.md in the root folder of the repository.

Further References

Arcadia Math relies on Arcadia Ring 2.
