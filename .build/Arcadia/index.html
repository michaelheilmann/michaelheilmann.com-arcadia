Project Arcadia
Description

Project Arcadia is an open environment for creating, regenerating, and preserving games. Its mission is to give developers, artists, and tinkerer's a future-proof space where games aren't just built - they are understood, reborn, and kept alive.

Arcadia is designed to be

  • portable and flexible - able to run across platforms while still embracing the unique strenghts of quirks of each one.
  • expressive and human-readable - games are described through clear, editable text documents that define their visuals, audials, and logics
  • procedurally alive - using deterministic procedural generation, Arcadia can recreate rich game worlds, sounds, and behaviours from compact descriptions
  • built for preservation - making it possible to store, regenerate, and study games long into the future
  • self-describing and self-publishing - Arcadia generates a complete deployable HTML5 website containing its own documentation ready for static web hosts or offline-reading

At its heart, Arcadia is about turning games into living artifacts: small, shareable, regenerable, and never truly lost.
