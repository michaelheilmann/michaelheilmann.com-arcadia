Michael Heilmann's Arcadia Visuals

This is the documentation for Michael Heilmann's Arcadia UI. Arcadia UI provides a user interface (windows, buttons, text fields, etc.) for Michael Heilmann's Arcadia Engine.

UI Basics: A user interface consists of a UI canvas node in which UI widgets are placed. The UI canvas node has a width and height, which may change, depending on the context in which the UI canvas is used. For example, a UI canvas might resize according to the application's window inner size. A UI canvas / its contained UI controls should react reasonably to these size changees.
