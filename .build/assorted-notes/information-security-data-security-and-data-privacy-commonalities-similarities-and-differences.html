Information Security, Data Security, and Data Privacy - Commonalities, similarities, and differences

Relation of information security and data security

The short version

What is the relation between information security and data security?

Data security protects digital data; information security protects all information, and data security lives inside it like a sub-discipline.

Relation of information security and data privacy

The short version

What is the relation of "information security" and "data privacy"?

Data privacy is about how personal data is collected, used, shared, and governed.

Information security is about protecting data including personal data from unauthorized access or damage. You cannot ensure privacy if you cannot ensure security.

Privacy decides what should happen with personal data. Security ensures only the right things can happen.

They overlap, but they are not the same.
