2025-12-24

Data Definition Language Schema specification was updated to version 0.2 This new version includes the Choice node: A DDLS choice nodes validates a DDL node against a list of DDLS nodes. If one DDLS node accepts, the DDLS choice node accepts. Otherwise the DDLS choice node rejects.

The prebuilt dependencies for Windows for Michael Heilmann's Arcadia have been updated. The following dependencies are now prebuilt:

  • flac
  • freetype
  • libpng
  • libressl
  • ogg
  • openal-soft
  • opengl
  • zlib

The binary archive is available as a release.

2024-02-11

As the backing system for the website matures, we add more documentation items for the various libraries. As of today, over 250 procedures, objects, and enumeration were documented.

When visiting this website (or running the website locally on your machine for quickly referencing the documentation), then you want that website to respect your privacy. In particular, you do not want that website to give your PI to companies by requesting assets and libraries from their CDNs: Consequently, this webiste removed most external references for to assets and libraries it uses. The following references were removed:

  • Removed references go Google's "static.google.com" for fonts. Fonts are hosted locally.
  • Removed references to "polyfill.io" and "cdn.jsdelivr.net" for library loading. The "Polyfill" library was removed and the "MathJax" library is hosted locally.

Currently, only the CDN reference Mathjax remains, which will be adressed shortly.
